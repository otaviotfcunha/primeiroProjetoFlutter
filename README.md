# primeiroapp - Primeiro Projeto em Flutter

Primeiro projeto em flutter, para aprendizagem e teste do ambiente de desenvolvimento.

## Começando

É um projeto de demonstração do flutter com algumas pequenas alterações.

As modificações que realizei foram:

- Acrescentar uma nova mensagem de texto;
- Variar as mensagens de texto de acordo com a quantidade de cliques do usuário;
- Mudar as cores dos textos conforme o usuário vai aumentando os cliques.

Todo o desenvolvimento contido aqui é para aprendizagem, para maiores informações, entre em contato comigo!
